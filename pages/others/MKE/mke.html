1.3 四种模式使用方法

  1. 清洁焕肤，先搭配仪器做导出清洁，再使用洗面奶洁面（无需搭配仪器）。

    深层清洁将毛孔深层污垢导出至肌肤表面，再使用洗面奶将表面污垢洗净。

  2. 每个模式5分钟内无操作后便自动关机，请按使用方法使用，切勿过度使用。

  3. 使用时需要握住机身两侧离子感应条并用仪器头接触肌肤，形成回路，使用时请保持肌肤湿润，否则仪器无法正常工作。

  4. 如需用于他人面部，需要与使用者接触肢体接触形成回路，否则仪器无法正常工作。

  5. 请按说明规定时长使用，切勿过度使用。

1.4 清洁焕肤模式使用方法

  1. 需搭配卸妆水/清洁型化妆水和化妆棉使用。

  2. 此模式能有效清洁毛孔中的污垢使肌肤透亮。

  3. 使用频率：1周2-3次，敏感肌可适当减少使用次数。

  4. 使用时长：不超过4分钟/次，敏感肌1-2分钟/次

1.5 光疗嫩肤模式使用方法

  1. 需搭配爽肤水、乳液、精华使用。

  2. 此模式能增强细胞活性，淡化皱纹并帮助护肤品深层滋润皮肤。

  3. 使用频率：可每日早晚各使用1次，敏感肌可适当降低使用频率。

  4. 使用时长：不超过4分钟/次，敏感肌1-2分钟/次，护肤品吸收即可停止使用，切勿干推。

1.6 温感提拉模式使用方法

  1. 需搭配面膜使用。

  2. 此模式能温和唤醒及放松肌肤，帮助肌肤恢复光彩和活力。

  3. 使用频率：可每日使用。

1.7 眼部护理模式使用方法

  1. 需搭配眼霜或眼部精华使用。

  2. 此模式可充分导入眼霜或眼部精华，促进眼霜或眼部精华吸收，从而达到眼部美容的效果。

  3. 使用频率：可每日早晚各使用1次，敏感肌可适当降低使用频率。

  4. 使用时长：不超过4分钟/次，敏感肌1-2分钟/次，眼霜或眼部精华吸收即可停止使用，切勿干推。

1.8 推荐使用方法

1.9 推荐护理步骤（水光肌养成记）

  1. 晨间醒肤模式

    唤醒肌肤 + 舒缓水肿 + 贴服妆容

    • 清洁面部
    • 涂抹爽肤水、精华和乳液
    • 选择光疗嫩肤模式
    • 上妆

  2. 晚间修复模式

    促进肌肤吸收 + 强润滋养 + 彻夜修复

    • 深层清洁（清洁焕肤模式，一周2-3次）
    • 清洁面部
    • 涂抹爽肤水、精华和乳液（光疗嫩肤模式）
    • 敷面膜导入（温感提拉模式）
