  1. react-router和react-router-dom

    react-router：提供了一些router的核心api，包括Router，Route，Switch等

    react-router-dom：提供了BrowserRouter，HashRouter，Route，Link，NavLink

  2. 路由模块安装

    npm install react-router-dom --save

    yarn add react-router-dom

  3. react-router-dom核心用法

    • HashRouter和BrowserRouter

      HashRouter ⇒ http://localhost:3000/#/admin/buttons

      BrowserRouter ⇒ http://localhost:3000/admin/buttons

    • Route：path、exact、component、render

      如：<Route path='/admin/ui/buttons' component={BUttons} />

      <Route path="/admin" render={()=> <Admin> <Route path='/admin/home' component={Home} /> </Admin> } />

    • NavLink、Link

      import {Link} from 'react-router-dom' const Header = () => ( <header> <ul> <li><Link exact={true} to='/'>Home</Link></li> <li><Link to='/about'>about</Link></li> <li><Link to='/three'>three</Link></li> </ul> </header> ) <Link to={{pathname: '/three/7'}}>Three #7</Link>

      定义：<Routr path="/three/:number" />

      取值：this.props.match.params.number

      // 一个基本的location对象 {pathname: '/',search: '', hash: '', key:'abc123', state: {}}

    • Switch

      <Switch> <Route path='/admin/ui/buttons' component={BUttons} /> <Route path='/admin/ui/modals' component={Modals} /> <Route path='/admin/ui/loading' component={Loading} /> </Switch>

    • Redirect

      路由重定向：<Redirect to="/admin/home" />
