第一章 集合与函数的概念

1.1 集合

  1. 列举法：{元素1，元素2...}

    描述法：E={x∈z | x=2k+1, k∈z}

  2. 集合关系：

    （1）A⊆B或B⊇A ⇒ A含于B或B包含A，A为B的子集。

    （2）A=B ⇒ 集合A与集合B相等。

    （3）A⊊B ⇒ A是B的真子集。

    （4）Ø ⇒ 空集，空集是任何集合的子集。

  3. 集合运算：

    A∪B ⇒ A与B的并集，A∪B={x|x∈A，或x∈B}

    A∩B ⇒ A与B的交集，A∩B={x|x∈A，且x∈B}

    U ⇒ 全集

    ∁uA ⇒ 集合A，由全集U中不属于A的所有元素组成的集合为A相对于U的补集。

    ∁uA={x|x∈U，且x∉A}

1.2 函数及其表示

  1. 函数：y=f(x)

    x --> 自变量，其取值范围A为函数的定义域。

    y --> 函数值，其集合{f(x)|x∈A}叫函数的值域。

  2. 区间

    闭区间 --> [a,b]

    开区间 --> (a,b)

    半开半闭区间 --> [a,b) , (a,b]

    端点 --> a、b

  3. 分段函数

  4. 映射：f:A-->B为集合A到集合B的一个映射。

1.3 函数的基本性质

  1. 增函数

  2. 减函数

  3. 最大值

  4. 最小值

  5. 偶函数 --> f(-x)=f(x)

  6. 奇函数 --> f(-x)=-f(x)

第三章 函数的应用

3.1 函数与方程

  1. ax2+bx+c=0（a≠0）→ 一元二次方程

    y=ax2+bx+c（a≠0）→ 二次函数

    判别式 Δ=b2-4ac

    （1）Δ>0时，一元二次方程有两个不等实根 x1、x2

    二次函数图像与x轴有两个交点（x1，0）、（x2，0）

    （1）Δ=0 ⇒ x1=x2，（x1，0）

    （2）Δ<0 ⇒ 无实根，与x轴无交点

  2. y=f(x)，使 f(x)=0 的实数x叫y=f(x)的零点

  3. 方程 f(x)=0 有实根

    → 函数 y=f(x) 图像与x轴有交点

    → y=f(x) 有零点

  4. 若函数 y=f(x) 在区间 [a,b]上的图像是连续不断地的一条曲线，且有f(a)·f(b)<0，那么 y=f(x) 在区间 (a,b)内有零点，即存在 c∈(a,b)，使得 f(c)=0。这个 c 为方程 f(x)=0 的根。

  5. 二分法：

    对于区间 [a,b] 上连续不断，且 f(a)·f(b)<0 的函数 y=f(x)，通过不断把函数 f(x) 的零点所在区间一分为二，逐步逼近零点。

3.2 函数模型及其应用

  1. 在区间 (0,+∞) 上，y=ax(a>1)、y=logax(a>1) 和 y=xn(n>0) 均为增函数，但增长速度不同。随着x的增大，y=ax(a>1) 增长速度越快，会越过并远远大于 y=xn(n>0) 的增长速度，而y=logax(a>1) 的增长速度会越慢。

    因此，总存在一个 x0，当 x>x0时，⇒ logax < xn < ax

第一章 空间几何体

1.1 空间几何体的结构

1.2　空间几何体的三视图和直观图

  1. 三视图：正视图、侧视图、俯视图

  2. 斜二测画法：

    （1）在已知图形中取互相垂直的x轴和y轴，两轴相交于点O，画直视图时，把它们画成对应的x'轴与y'轴，两轴交于点O'，且使∠x'O'y'=45°（或135°），它们确定的平面表示水平面。

    （2）已知图形中平行于x轴或y轴的线段，在直视图中分别画成平行于x'轴或y'轴的线段。

    （3）已知图形中平行于x轴的线段，在直视图中保持原长度不变，平行于y轴的线段，长度为愿来的一半。

  3. 平行投影、中心投影

1.3 空间几何体的表面积与体积

  1. 圆柱表面积：

    S=2πr²+2πrl=2πr(r+l)

    r→圆柱底面半径，l→圆柱母线长，πr²→圆柱底面面积，2πrl→圆柱侧面面积

  2. 圆锥表面积：

    S=πr²+πrl=πr(r+l)

  3. 圆台表面积：

    S=π(r'²+r²+r'l+rl)

  4. 圆柱体积：

    V=Sh （S→底面面积，h→高）

    棱柱体积也是：V=Sh

  5. 圆锥体积：V=(1/3)Sh

    棱锥体积：V=(1/3)Sh

  6. 圆台（棱台）体积：

    V=(1/3)(S'+√(S'S)+S)h

    S'、S分别为上下底面积，h→高

  7. 球的体积：

    V=(4/3)πR³

  8. 球的表面积：

    S=4πR²

第二章 点、直线、平面之间的位置关系

2.1　空间点、直线、平面之间的位置关系

  1. 如果一条直线上的亮点在一个平面内，那么这条直线在此平面内。

  2. 过不在一条直线上的三点，有且只有一个平面

  3. 如果两个不重合的平面在平面有一个公共点，那么它们有且只有一条过该点的公共直线。

  4. 共面直线 → 相交直线 → 同一平面内，有且只有一个公共点

    共面直线 → 平行直线 → 同一平面内，没有公共点

    异面直线 → 不同在任何一个平面内，没有公共点

  5. 平行于同一条至咸宁的两条直线互相平行

  6. 空间中如果两个角的两边分别对应平行，那么这两个角相等或互补。

  7. 如果两条异面直线所成的角是直角，那么这两条直线互相垂直。如a⊥b

  8. 直线与平面的位置关系：

    （1）直线在平面内

    （2）直线与平面相交

    （3）直线与平面平行

  9. 两个平面间的位置关系：

    （1）两个平面平行

    （2）两个平面相交

2.2　直线、平面平行的判定及其性质

  1. 平面外一条直线与此平面内的一条直线平行，则该直线与此平面平行。

  2. 一个平面内的两条相交直线与另一个平面平行，则这两个平面平行。

  3. 一条直线与一个平面平行，则过这条直线的任一平面与此平面的交线与该直线平行。

  4. 如果两个平行平面同时和第三个平面相交，那么它们的交线平行。

2.3　直线、平面垂直的判定及其性质

  1. 如果直线l与平面α内的任意一条直线垂直，则直线l与平面α互相垂直，记作l⊥α。直线l叫做平面α的垂线，平面α叫做直线l的垂面。直线与平面垂直时，它们唯一的公共点P叫做垂足。

  2. 一条直线与一个平面内的两条相交直线都垂直，则该直线与此平面垂直。

  3. 从一条直线出发的两个半平面所组成的图形叫做二面角。这条直线叫做二面角的棱，这两个半平面叫做二面角的面。

  4. 二面角的平面角

  5. 一般地，两个平面相交，如果他们所成的二面角是直二面角。则这两个平面互相垂直。

  6. 一个平面过另一个平面的垂线，则这两个平面垂直。

  7. 垂直于同一个平面的两条直线平行。

  8. 两个平面 垂直，则一个平面内垂直于交线的直线与另一个平面垂直。
