言语

1、中心理解

转折关系、主题词、因果关系、必要条件关系、程度词、并列关系、行文脉络、细节判断
【提问方式】
  • 这段文字主要 / 旨在 / 重在 / 意在 / 想要说明（论述、强调）的是……
  • 这段文字的主旨 / 主题 / 观点是……
  • 对这段文字概括最恰当的一项是……
  • ......
【解题思路】
  • 1. 重点词语
  • 2. 行文脉络
【重点词语之关联词 - 转折关系】

  • * 典型标志词：

    • 虽然……但是……
    • 尽管……可是……
    • ……不过……
    • ……然而……
    • ……却……
    • 其实 / 事实上 / 实际上
    • ......

  • * 错误选项特征

    • 1. 转折前的内容
    • 2. 围绕例子的表述
    • 3. 围绕原因的表述
    • 4. 无中生有
    • 略读句子特征：例子、原因……

  • * 总结

    • 1. 转折关系（转折之后是重点）
    • 2. 典型标志词：虽然...但是...； 尽管...可是...； 不过； 然而； 却； 其实/事实上/实际上
    • 3. 非典型标志词：殊不知； 截然不同； 截然相反； 全新的研究； 一种误读； 相对而言；
    • 4. 错误选项特征：转折前的内容、围绕例子的表述、围绕原因的表述、无中生有
    • 略读句子特征：例子、原因......
【重点词语 - 主题词】

  • *

    • 1. 文段围绕其展开
    • 2. 一般高频出现
    • 3. 多为名词
    • 4. 前有引入或后有解释说明

  • * 总结

    • 1. 主题词
    • 2. 特征：文段围绕其展开；一般高频出现； 多为名词； 前有引入或后有解释说明；
    • 3. 理论要点：正确选项中需包含文段主题词
    • 4. 干扰选项：范围扩大/缩小/偷换
【因果关系】

  • * 典型标志词：

    • 因为……所以……
    • 由于……因此……

  • * 理论要点：结论是重点：

    • 引导结论的标志词：
    • 1. 所以、因此、因而、故而、于是、可见、看来（为什么、为何）
    • 2. 导致、致使、使得、造成

  • * 知识链接：

    • 结论句出现在文段开头 / 中间，之后仍有其他语句
    • 1. 之后的语句是进一步解释说明，此时中心句仍为结论句
    • 2. 之后又出现并列、因果、转折、对策等，需结合多种关联词共同分析

  • * 总结

    • 1. 因果关系（结论是重点）
    • 2. 典型标志词：因为...所以...； 由于...因此...
    • 3. 因果标志词：所以、因此、因而、故而、于是、可见、看来（为什么、为何）      导致、致使、使得、造成
    • 4. 文段特征：结论句在结尾（多为文段中心句）；      结论句在开头/中间（后为进一步说明 - 结论句为中心句； 后有其他关联关系 - 需结合多种关联词共同分析）
【必要条件关系】

  • * 典型格式：

    • 只有……才……

  • * 理论要点：

    • 必要条件是重点
    • 必要条件即“只有”和“才”之间的部分

  • * 知识链接：对策常考，很重要

    • 对策标志词：
    •   1. 应该、应当、必须、需要、亟须、亟待 + 做法
    •   2. 通过 / 采取……手段 / 途径 / 措施 / 方式 / 方法 / 渠道，才能……
    •   3. 呼吁、倡导、提倡、提醒、建议 + 做法
    •   4. 前提、基础、保障 、关键
    • 行文脉络：
    •   1. 提出问题 + 分析问题 + 解决问题
    •   2. 提出问题 + 解决问题 + 解释说明（意义效果）
    •   3. 对策 + 正反论证 / 原因论证
    • 反面论证：
    •   典型格式：如果 / 倘若 / 一旦……+ 不好的结果
    •   具体应用：把前面的做法反过来，即为正确答案
    • 当文段中只有“提出问题 + 分析问题”的部分时，“解决问题”通常会作为正确答案出现在选项里

  • * 总结

    • “只有...才”中间是重点
    • 1. 典型格式：只有...才...
    • 2. 对策标志词：应该、应当、必须、需要、亟须、亟待+做法      通过/采用...手段/途径/措施/方式/方法/渠道，才能...      呼吁、倡导、提倡、提醒、建议+做法      前提、基础、保障、关键
    • 3. 文段特征：对策在结尾（提出问题+分析问题+解决问题）；      对策在中间（提出问题+解决问题+解释说明  （意义效果））      对策在开头（对策+正反论证/原因论证）
    • 4. 反面论证：典型格式（如果/倘若/一旦...+不好的结果）；      具体应用（把前面的做法反过来，即为正确答案）
    • 5. 知识链接：文段中只出现“提出问题+分析问题”，“解决问题”可能出现在选项中作为正确答案；      对策不万能，一定要注意
【程度词】

  • * 标志词：

    • 更、尤其、正是、特别是、真正、根本、最（核心、突出）等

  • * 理论要点：

    • 程度词所在的语句通常为重点

  • * 总结

    • 1. 标志词：更、尤其、正是、特别是、真正、根本、最（核心、突出）等
    • 2. 理论要点：程度词所在的语句通常为重点
【并列关系】

  • * 理论要点：

    • 概括全面完整

  • * 文段特征：

    • 1. 包含并列关联词及标点，如此外、另外、同时、以及、“ ；”
    • 2. 句式相同或相近
    • 3. 无明显其他关联词语

  • * 选项特征：

    • 1. 两方面情况：和、及、与、同
    • 2. 更多种情况：许多、一些、不同、各种、一系列

  • * 总结

    • 1. 概括全面完整
    • 2. 文段特征：包含并列关联词及标点，如此外、另外、同时、以及、“；”；      句式相同或相近；      无明显其他关联词语
    • 3. 选项特征：两方面情况：和、及、与、同；      更多种情况：许多、一些、不同、各种、一系列
    • 4. 提示：表述片面的选项需要排除
【行文脉络】

  • * 理论要点：

    • 把握中心句及分述句的特征

  • * 中心句特征：

    • 观点（对策、结论、评价）

  • * 分述句特征：

    • 1. 举例子：“比如”“例如”“……就是例证”等
    • 2. 调查报告、数据资料等
    • 3. 正反论证
    • 4. 原因解释
    • 5. 并列分述

  • * 知识链接：总—分（观点 + 解释说明）

    • 总—分（观点 + 解释说明）
    •   中心句后为不同类型的解释说明
    • 分—总
    •   1. 结论、对策
    •   2. 代词（“这”“此”）引导的尾句需关注

  • * 总结

    • 1. 理论要点：把握中心句及分述句的特征
    • 2. 中心句特征：观点（对策、结论、评价）
    • 3. 分述句特征：
    • 4. 提示：选项陷阱总结：非重点，迷你眼      无中生有不能选
【程度词】

  • * 典型标志词：

    • 因为……所以……

  • * 理论要点：结论是重点：

    • 引导结论的标志词：
    • 1. 所以、因此、因而、故而、于是、可见、看来（为什么、为何）

  • * 知识链接：

    • 1. 之后的语句是进一步解释说明，此时中心句仍为结论句
    • 2. 之后又出现并列、因果、转折、对策等，需结合多种关联词共同分析

  • * 总结

    • 1. 因果关系（结论是重点）
    • 2. 典型标志词：因为...所以...； 由于...因此...
    • 3. 因果标志词：所以、因此、因而、故而、于是、可见、看来（为什么、为何）      导致、致使、使得、造成
    • 4. 文段特征：结论句在结尾（多为文段中心句）；      结论句在开头/中间（后为进一步说明 - 结论句为中心句； 后有其他关联关系 - 需结合多种关联词共同分析）

2、细节判断题

【提问方式】
  • 以下对文段理解正确 / 不正确的是……
  • 符合 / 不符合这段话意思的是……
  • 从文段中可以得知 / 推出的是……
  • ......
【典型细节题】

  • * 错误选项类型：

    • 1. 无中生有
    • 2. 偷换概念（替换、混搭）
    • 3. 偷换时态
    •   将来时（将要、立刻、趋势、以后）
    •   完成时（已、已经、了、完成）
    •   进行时（正在、在……中、着）
    • 4. 偷换逻辑

  • * 快速解题技巧

    • 1. 对比项
    •   标志词：A 比 B 更…… 、 A 高于 / 优于 B 、 利大于弊 、 ...
    • 2. 相对绝对项
    •   绝对表述：一定、必定、都……
    •   相对表述：可能、也许、往往……
    • 3. 表述与实际不符项
【细节主旨化】

  • * 理论要点：

    • 优选契合主旨的选项
【细节判断 - 总结】
  • 1、提问方式：
  •   （1）以下对文段理解正确/不正确的是...
  •   （2）符合/不符合这段话意思的是...
  •   （3）从文段中可以得知/推出的是...
  • 2、错误选项类型：
  •   （1）无中生有
  •   （2）偷换概念（替换、混搭）
  •   （3）偷换时态
  •     - 1）将来时（将要、立刻、趋势、以后）
  •     - 2）完成时（已、已经、了、完成）
  •     - 3）进行时（正在、在...中、着）
  •   （4）偷换逻辑
  • 3、快速解题技巧：对比项、相对绝对项、表述与实际不符项
  • 4、细节主旨化：优选契合主旨的选项

3、语句排序题

【提问方式】
  • 将以上 / 以下几个句子重新排列，语序正确的是……
【理论要点】
  • 1. 根据选项提示，对比后确定首句
  • 2. 确定捆绑集团 / 确定顺序 / 确定尾句
  • 3. 验证（只验证你基本锁定的答案，而非全部验证
【确定首句】

  • * 理论要点：

    • 1、下定义（……就是 / 是指）
    • 2、背景引入（随着、近年来、在……大背景 / 环境下）
    • 3、非首句特征（指代词、关联词后半部分）
【确定捆绑集团】

  • * 理论要点：

    • 1、指代词捆绑
    • 2、关联词捆绑
【确定顺序】

  • * 理论要点：

    • 1、时间顺序
    • 2、逻辑顺序
【确定尾句】

  • * 理论要点：

    • 因此、所以、看来、于是、这、应该、需要
【语句排序题 - 总结】
  • 1、提问方式：将以上 / 以下几个句子重新排列，语序正确的是……
  • 2、解题要点：从选项入手（确定首句、确定捆绑集团/确定顺序/确定尾句）
  • 3、确定首句
  •    首句特征（下定义：...就是/是指； 背景引入：随着、近年来、在...大背景/环境下）
  •    非首句特征（指代词：这、那、他、其、该、此； 关联词后半部分：但、却、而且）
  • 4、确定捆绑集团
  •   指代词：这、那、他、其、该、此
  •   关联词：配套出现确定捆绑 、 单个出现分析句意
  • 5、确定顺序：时间顺序、逻辑顺序（观点+解释说明 、 A和B）
  • 6、确定尾句：
  •   因此、所以、看来、于是、这...引导的总结性尾句
  •   对策

4、语句填空题

【提问方式】
  • 填入画横线部分最恰当的一句 / 项是……
【理论要点】
  • 横线在结尾：1. 总结前文; 2. 提出对策
  • 横线在开头：需概括文段的中心内容
  • 横线在中间：1. 注意与上下文的联系; 2. 把握好主题词，保证与文段话题一致
【语句填空题 - 总结】
  • 1、提问方式：填入画横线部分最恰当的一句 / 项是……
  • 2、横线在结尾：总结前文 、 提出对策
  • 3、横线在开头：需概括文段的中心内容
  • 4、横线在中间：注意与上下文的联系 、 把握好主题词，保证与文段话题一致

5、接语选择题

【提问方式】
  • 作者接下来最有可能讲述的是……
【理论要点】
  • 重点关注文段最后一句话
  • 干扰项特征：文段中已经论述过的内容
【接语选择题 - 总结】
  • 1、提问方式：作者接下来最有可能讲述的是……
  • 2、理论要点：重点关注文段最后一句话
  • 3、干扰项特征：文段中已经论述过的内容
  • 4、提示：理解要紧紧依托文段，不能无中生有

5、逻辑填空 - 词的辨析

【提问方式】
  • 作者接下来最有可能讲述的是……
【词义侧重】

  • * 辨析方法：

    • 1. 用不一样的字组词
    • 2. 整体进行搭配组词
【固定搭配】

  • * 理论要点：

    • 找准搭配对象（常用词、热点词）
【程度轻重】

  • * 理论要点：

    • 所填词语的程度轻重与文段意思的轻重保持一致
【感情色彩】

  • * 理论要点：

    • 1. 褒义、贬义、中性
    • 2. 所填词语的感情色彩与文段的感情色彩保持一致
【词的辨析 - 总结】
  • 1、词义侧重，辨析方法：
  •   （1）第一种：拆字组词：用不一样的字单独组词
  •   （2）第二种：整词搭配：用整个词进行组词搭配
  • 2、固定搭配：
  •   （1）搭配一个词：瞻前顾后找准搭配对象
  •   （2）搭配多个词：横线所填词语搭配有“和” “及” “与”引导的并列结构 ； 需与并列结构搭配恰当
  • 3、程度轻重：
  •   （1）区分词的程度轻重
  •     - 程度轻：大相径庭：相差很大，相差很多
  •     - 程度重：截然不同：完全相反，完全不同
  •   （2）理论要点：所填词语的程度轻重与文段意思的轻重保持一致
  • 4、感情色彩
  •   （1）区分词的感情色彩（褒义-成果 ； 贬义-后果 ； 中性-结果）
  •   （2）理论要点：所填词语的感情色彩与文段的感情色彩保持一致

5、逻辑填空 - 语境分析

【提问方式】
【关联关系】

  • * 转折：

    • 理论要点：前后语义相反

  • * 递进：

    • 理论要点：语义程度前轻后重

  • * 并列：

    • 理论要点：
    • 1. 同义并列：顿号（、）、逗号（，）
    • 2. 反义并列：不是……而是……、是……不是……、相反、反之等
【对应关系】

  • * 解释类对应：

    • 题干特点：分句，______，分句
    • 标志词：即、就是、可以说、可谓、无异于、比如、例如等
    • 标点：冒号（ ：）、破折号（——）

  • * 重点词句对应：

    • 1. 重点词：指代词 / 主题词 / 形象表达
    • 2. 重点句：完整语句

  • * 提示：

    • 文段中的完整语句可作为解题的提示

  • * 解释类对应：

    • aaaaaaaaaaaaaaaaaaaaa

  • * 解释类对应：

    • aaaaaaaaaaaaaaaaaaaaa
【语境分析 - 总结】
  • 1、关联关系：
  •   （1）转折：标志词（但是、然而、却等）、理论要点（前后语义相反）
  •   （2）递进：标志词（而且、甚至、更等）、理论要点（语义程度前轻后重）
  •   （3）并列：标志词（同义并列、反义并列）、理论要点（同义并列连接前后语义相近；反义并列连接前后语义相反）、提示（相同句式也可表示并列）
  •     - 1）标志词：同义并列（顿号（、）、逗号（，））、反义并列（不是……而是……、是……、不是……、相反、反之等）
  •     - 2）理论要点：同义并列连接前后语义相近；反义并列连接前后语义相反
  •     - 3）提示：相同句式也可表示并列
  • 2、对应关系：
  •   （1）解释类对应
